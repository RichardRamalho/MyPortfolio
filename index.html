Tecnologias

Confira as tecnologias que mais utilizo:

HTML

HTML é a linguagem fundamental para criar páginas web. Com uma sintaxe simples, ela permite estruturar conteúdo, inserir mídia e criar links

CSS

CSS é uma linguagem para estilizar elementos HTML em páginas web, controlando cores, fontes e layout

JavaScript

Uma das linguagens de programação mais utilizadas hoje em dia

React.Js

Biblioteca usada para construir interface de usuário(UI)

Node.Js

Com essa tecnologia podemos criar desde sites até APIs diretamente do servidor

*ainda estou desenvolvendo

Git

Git é um sistema de controle de versão usado para rastrear alterações no código-fonte. Facilita a colaboração entre desenvolvedores e o gerenciamento de diferentes versões do código

GitHub

GitHub é uma plataforma de hospedagem de código-fonte baseada em nuvem que utiliza o sistema de controle de versão Git

Figma

Figma é uma ferramenta de design colaborativo baseada na nuvem, usada por designers para criar e prototipar interfaces de usuário para aplicativos e websites.
